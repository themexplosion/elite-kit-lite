== Description ==

Take your [Elementor](https://wordpress.org/plugins/elementor/) page building experience to the next level with amazing elements and extensions. Edit websites easily and professionally with easy-to-use elementor addons.

### Fully Customizable
Each and every element is completely customizable. Every text, color, hover color etc. can be customized in such a way, you can achieve almost any design you want.

### Light Weight & Optimized
The plugin is optimized for better performance. We follow the best practices to optimize the website, as much as possible.

== Installation ==

Note : Elementor plugin is mandatory to use this plugin. Make sure you have [Elementor](https://wordpress.org/plugins/elementor/) installed.


1. Upload the plugin folder to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Elite Kit Addons will appear at the top in Elementor Widget panel, when editing the webpage
4. Also you can see them under the category "Elite Kit Widgets" on your element/widget list.

== Frequently Asked Questions ==

= Can I use the plugin without Elementor Page Builder? =

No. You cannot use without Elementor since it's an addon for Elementor.

= Does it work with any theme? =

Of course! It will work with any theme where Elementor works.
